# Overview

## Important

Make sure pickle is git ignored or can break version control if large enough

## Setup

Requires Python 3.13+ and [uv](https://docs.astral.sh/uv/).

```bash
uv sync
```

This installs all dependencies (`gymnasium`, `numpy`) into a local virtual environment. To run a script:

```bash
uv run <script.py>
```
